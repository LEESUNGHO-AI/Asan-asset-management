2. `index.html` 파일을 드래그앤드롭으로 업로드

1. 저장소에서 기존 `index.html` 삭제
2. 새 `index.html` 업로드
3. 자동 배포 (1-2분 소요)

## 📊 대시보드 기능

- **KPI 현황**: 총 자산(85개), 총 가치(₩152억), 가동률, 예산 집행률
- **자산 목록**: Notion DB 실제 데이터 표시 (클릭시 Notion으로 이동)
- **차트**: 카테고리별/담당자별 분포
- **인프라 프로젝트**: 5개 대형 프로젝트 진행 현황
- **예산 현황**: 집행률 및 4분기 계획
- **리스크 현황**: Critical/High/Medium 등급별 관리
